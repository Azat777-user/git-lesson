Commands git:

  • git clone path to repository - clone repository from github
  • git status - status of the current local repository
  • git add file name if it is in the root directory otherwise the file path - add a new file under the control of the git
  • git commit -am "added new file index.html" - add comments(change clamp)
  • git push - send all committed changes to the repository to the server
  • git pull - pick up fresh updates
  • git branch - Lists out all the branches
